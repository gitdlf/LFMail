# LFMail
利用JavaMail与百度语音实现的邮件播报

# How to use

类路径下配置 emailServerConfig.properties

POP 邮件服务器的地址
mailServer_POP3Host = pop.163.com

SMTP 邮件服务器的地址
mailServer_SMTPHost = smtp.163.com

邮箱地址
myEmailAddress = xxxxxx@163.com

邮箱用户名
userName = xxxxxx

注意使用的是开通 SMTP、 POP、IMAP 协议的授权码
password = xxxxxx

是否需要验证
validate = true
